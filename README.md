# CoinFlip — strony prawne (GitHub Pages)

Publiczne dokumenty wymagane przez Google Play i App Store.

## Włącz GitHub Pages

1. Repo → **Settings** → **Pages**
2. **Source:** Deploy from a branch
3. **Branch:** `main` / folder **/ (root)**
4. Zapisz — po 1–2 min strony będą pod:

- https://s0ul3r.github.io/coinflip-legal/
- https://s0ul3r.github.io/coinflip-legal/privacy.html
- https://s0ul3r.github.io/coinflip-legal/delete-account.html
- https://s0ul3r.github.io/coinflip-legal/terms.html

Te URL-e są ustawione w aplikacji CoinFlip (`EXPO_PUBLIC_PRIVACY_POLICY_URL`, `EXPO_PUBLIC_ACCOUNT_DELETION_URL`).
